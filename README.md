## Blog Admin Express app

A blog application which uses Node.js, Express, MongoDB, Passport, Google OAuth strategy to manage the posts of a logged user.

## Execute

#### `npm run start`
#### `npm run dev`
#### `npm test
#### `npm build`

## Tech Stack
-
- [x] Express
- [x] Mongoose
- [x] Handlebars
- [x] Moment
- [x] Morgan
- [x] Passport
- [x] Passport-google-oauth20
